## **1. Time and Work Basics**

- **Work** is assumed to be **1 unit** (the whole task).
    
- If a person can finish the work in `N` days, then:
    
    Work done in 1 day=1N\text{Work done in 1 day} = \frac{1}{N}Work done in 1 day=N1​
- **Efficiency** is the amount of work done in 1 day.

## **2. Example 1 – Wall Building with A, B, and C**

**Given:**

- A can build in **10 days** → 110\frac{1}{10}101​ work/day
    
- B can build in **5 days** → 15\frac{1}{5}51​ work/day
    
- C can destroy in **20 days** → −120-\frac{1}{20}−201​ work/day
    

**Net Rate:**

110+15−120\frac{1}{10} + \frac{1}{5} - \frac{1}{20}101​+51​−201​

Take LCM = 20:

220+420−120=520=14\frac{2}{20} + \frac{4}{20} - \frac{1}{20} = \frac{5}{20} = \frac{1}{4}202​+204​−201​=205​=41​

So they complete **1/4 of work per day**.

**Total Time:**

Time=Total workRate=11/4=4 days\text{Time} = \frac{\text{Total work}}{\text{Rate}} = \frac{1}{1/4} = 4\ \text{days}Time=RateTotal work​=1/41​=4 days

✅ **Answer:** **4 days**

## **3. Example 2 – Wage Distribution Between A and B**

**Given:**

- A can do work in **10 days** → 110\frac{1}{10}101​ work/day
    
- B can do work in **15 days** → 115\frac{1}{15}151​ work/day
    
- Total wage = ₹50
    

**Combined Rate:**  
LCM = 30

330+230=530\frac{3}{30} + \frac{2}{30} = \frac{5}{30}303​+302​=305​

So in 1 day: A = 330\frac{3}{30}303​, B = 230\frac{2}{30}302​.

**Ratio of Work Done:**  
A : B = 3 : 2

**Wage Split:**  
₹50 in 3 : 2 →  
A = ₹30, B = ₹20

✅ **Answer:** **B gets ₹20**
